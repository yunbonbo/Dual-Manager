# カレンダー連携（.ics ダウンロード）

予約を Google カレンダー、Outlook、Apple カレンダー等に追加するための .ics ファイルダウンロード機能です。

---

## 機能概要

- **予約完了時（顧客）**: 予約確定後に「カレンダーに追加（.ics）」ボタンでダウンロード
- **ダッシュボード（管理者）**: 各予約カードの「.ics」ボタンでダウンロード

ダウンロードした .ics ファイルをカレンダーアプリにインポートすると、予約がカレンダーに登録されます。

---

## 使い方

### 顧客向け

1. 予約を確定する
2. 「予約が確定しました！」画面で「カレンダーに追加（.ics）」をクリック
3. ダウンロードされた .ics ファイルを開く、またはカレンダーアプリにインポート

### 管理者向け

1. ダッシュボードで予約一覧を表示
2. 各予約カードの「.ics」ボタンをクリック
3. ダウンロードされた .ics ファイルを Google カレンダー等にインポート

---

## Google カレンダーへの追加方法

1. [Google カレンダー](https://calendar.google.com/) を開く
2. 右上の **⚙️ 設定** → **インポートとエクスポート**
3. **ファイルを選択** でダウンロードした .ics を選択
4. **インポート** をクリック

---

## 今後の拡張（Google カレンダー API 連携）

管理者の Google カレンダーへ自動登録するには、Google Calendar API と OAuth の設定が必要です。実装時は以下を検討してください。

- Google Cloud でプロジェクト作成・Calendar API 有効化
- OAuth 2.0 認証情報の取得
- 管理者が Google アカウントで連携を許可
- 予約作成・更新時に API でイベントを登録
