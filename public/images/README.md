# 画像フォルダ

このフォルダに、予約フォームの店舗・メニューアイコン用の画像を配置してください。

## ファイル名のルール

### 店舗アイコン（「店舗を選択」のカード）

- **理容室**: `shop-理容室.png`（または `.jpg`, `.svg`）
- **美容室**: `shop-美容室.png`

※ `shops` テーブルの `name` カラムの値と一致させる必要があります。

### メニューアイコン（「メニューを選択」のカード）

- **カット+シャンプー**: `menu-カット+シャンプー.png`
- **カット+カラー**: `menu-カット+カラー.png`
- **パーマ+トリートメント**: `menu-パーマ+トリートメント.png`

※ `menus` テーブルの `name` カラムの値と一致させる必要があります。

## 推奨サイズ

- 店舗アイコン: 64×64px 以上
- メニューアイコン: 56×56px 以上

## 編集するファイル

画像のパスを変更する場合は **`pages/index.vue`** を編集してください。

- 店舗アイコン: 61行目付近の `:src="\`/images/shop-${shop.name}.png\`"`
- メニューアイコン: 97行目付近の `:src="\`/images/menu-${menu.name}.png\`"`
